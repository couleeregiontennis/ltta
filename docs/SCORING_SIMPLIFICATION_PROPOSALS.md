# 2026 Scoring Simplification Proposals

The current LTTA scoring system (10/8/6/4/2/1/0) is often a source of confusion for captains and players. Below are three simplified options to be presented at the April 14th meeting.

---

## Option 1: Simple Win/Loss (1/0)
Every match is worth 1 point.
*   **Match Win:** 1 Point
*   **Match Loss:** 0 Points
*   **Halted Match (5:30 PM):** 1 point to the team leading in games, 0.5 points each if tied.
*   *Philosophy:* Winning is all that matters. Extremely easy to track.

## Option 2: The "Performance" Scale (3/2/1/0)
Rewards teams for competitiveness even in a loss.
*   **Straight Set Win:** 3 Points
*   **Split Set Win:** 2 Points
*   **Split Set Loss (Taking a set):** 1 Point
*   **Straight Set Loss:** 0 Points
*   *Philosophy:* Encourages fighting for that second set even if you lose the third set tiebreak.

## Option 3: Raw Set Count (2/1/0)
Total points = Total sets won.
*   **Straight Set Win:** 2 Points (Winner), 0 Points (Loser)
*   **Split Set Win:** 2 Points (Winner), 1 Point (Loser)
*   *Philosophy:* The most accurate representation of play. If you win 2 sets, you get 2 points. If you win 1, you get 1.

---

## Handling Special Cases (Simplified)

### 1. Halted Matches (5:30 PM Time Limit)
If a match is unfinished at 7:00 PM:
*   **Option A (Games):** Team with the most games won gets the "Win."
*   **Option B (Split):** If teams have won 1 set each, it is recorded as a draw/tie (0.5 points in Option 1, or 1 point each in Option 3).

### 2. Forfeits & Retirements
*   **Forfeit:** Opponent gets Max Points (e.g., 3 points); Forfeiting team gets 0.
*   **Retirement:** Treated as a loss from the point of retirement. Opponent gets Max Points; Retired team keeps any sets/points earned up to that point.

### 3. Sanctioned Player Penalties
Instead of complex point deductions:
*   Using a higher-rated player results in an automatic **Forfeit (0 points)** for that line, while the opponent receives Max Points.
